# eslint-plugin-async
[![MIT licensed](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

> async/await linting rules for ESLint

## Install

```bash
$ npm install --save-dev eslint-plugin-async
```

## Configuration

Edit your `.eslintrc`

```json
{
  "plugins": ["async"],
  "rules": {
    "async/missing-await-in-async-fn": 1
  }
}
```
